# cocoapods

- dependency manager for swift

## installation

```bash
> sudo gem install cocoapods
```

### alamofire

- package used to call REST APIs
- developed in swift

```bash

# create xcode project
# close the project
# open the terminal and go to the project directory
# fire following commands

# initialize the podfile
> pod init

# open the podfile and add following line
> pod 'Alamofire'

# install the dependencies
> pod install

# once the dependencies are installed, open a file with an extension .xcworkspace
# DO NOT USE .xcodeproj file

# set the following property to No
# User Script Sandboxing -> No

```
